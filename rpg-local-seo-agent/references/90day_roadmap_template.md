## GBP Suspension Priority Checklist

Use this when a client's GBP is suspended:

- [ ] Identify suspension type (soft vs. hard)
- [ ] Gather: business license, insurance cert, utility bill at listed address
- [ ] Verify address matches website footer and all citations exactly
- [ ] If SAB: hide address, set service areas
- [ ] Submit appeal with all documentation
- [ ] Follow up at 2-week mark if no response
- [ ] If denied: fix flagged issues and resubmit
