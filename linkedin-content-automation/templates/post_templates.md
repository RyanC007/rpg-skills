# LinkedIn Post Templates

These templates provide structure for different types of LinkedIn posts.

## Standard Post Template

```
[HOOK - Attention-grabbing opening line or question]

[CONTEXT - Brief setup or problem statement]

[VALUE - Main insight, lesson, or actionable advice]
[Supporting detail or example]
[Additional context if needed]

[ENGAGEMENT QUESTION - Thought-provoking question to drive comments]

#Hashtag1 #Hashtag2 #Hashtag3 #Hashtag4 #Hashtag5
```

## Story-Based Post Template

```
[HOOK - Intriguing story opening]

[SITUATION - Set the scene]

[CHALLENGE - What went wrong or what was difficult]

[INSIGHT - What you learned or discovered]

[APPLICATION - How this applies to your audience]

[ENGAGEMENT QUESTION]

#Hashtags
```

## Tactical/How-To Post Template

```
[HOOK - Promise of value]

Here's how to [achieve result]:

1. [Step one with brief explanation]
2. [Step two with brief explanation]
3. [Step three with brief explanation]

[BONUS TIP or COMMON MISTAKE to avoid]

[ENGAGEMENT QUESTION - Ask about their experience or challenges]

#Hashtags
```

## Thought Leadership Post Template

```
[BOLD STATEMENT or CONTRARIAN TAKE]

Everyone talks about [popular approach].
But here's what they're missing: [your insight]

[SUPPORTING ARGUMENT 1]
[SUPPORTING ARGUMENT 2]
[SUPPORTING ARGUMENT 3]

[CONCLUSION - Tie it back to practical application]

[ENGAGEMENT QUESTION - Invite debate or perspective]

#Hashtags
```

```

[Brief context for why this matters]

Key takeaways:
→ [Point 1]

[Call to action or next step]

[ENGAGEMENT QUESTION]

#Hashtags

[Note: Attach infographic image]
```

## Question/Engagement Post Template

```
[SETUP - Brief context]

[MAIN QUESTION - Clear, specific question]

[OPTIONAL: Provide 2-3 options or perspectives]

[Why this matters / What you'll do with responses]

Drop your answer in the comments 👇

#Hashtags
```

## Formatting Guidelines

- Use line breaks generously for readability
- Keep paragraphs to 2-3 sentences max
- Use emoji sparingly (1-2 per post maximum)
- Bold or CAPS for emphasis (use sparingly)
- Use → or • for bullet points when needed
- Always end with an engagement question
- Include 3-5 relevant hashtags at the end
