# Supercharged SEO & AEO Research + 6-Month Content Plan - Master Project Prompt (v2)

**Purpose:** This prompt instructs Manus AI to conduct comprehensive SEO/AEO research and deliver a complete 6-month content strategy for any company in any niche, **with or without an existing website**. It includes technical SEO analysis for existing sites.

---

## 🎯 PROJECT INSTRUCTIONS

You are tasked with conducting **deep, comprehensive SEO and AEO (Answer Engine Optimization) research** for [COMPANY NAME], a [INDUSTRY/NICHE] business serving [PRIMARY GEOGRAPHIC AREA].

Your goal is to deliver a complete, actionable 6-month content strategy that will establish [COMPANY NAME] as the dominant digital authority in their market. This system is designed to handle both new businesses (no website) and established businesses with existing websites.

---

## 📋 RESEARCH METHODOLOGY (ENHANCED 15-PHASE SYSTEM)

Follow this exact methodology in sequence. Phases 13-15 are only for clients with existing websites.

### PHASE 1-12: Standard Research (For All Clients)
*(These phases are based on the original, proven system for foundational strategy)*

1.  **Business Context Analysis (30 mins):** Deeply understand the business, value props, target markets, and competitive positioning.
2.  **Keyword Research (2-3 hours):** Identify 100+ high-value keywords (transactional, informational, local, long-tail).
3.  **AEO Research (1-2 hours):** Research Answer Engine Optimization best practices and identify 50+ question-based opportunities.
4.  **Competitive Intelligence (3-4 hours):** Analyze 4-6 top competitors to identify their strategies, strengths, and weaknesses.
5.  **Strategic Recommendations (2 hours):** Synthesize all research into clear, actionable strategic pillars and content opportunities.
6.  **Comprehensive Strategy Document (3-4 hours):** Create the main `[Company]_SEO_AEO_Strategy.md` document.
7.  **Keyword Database (1 hour):** Create the `[Company]_Keyword_Database.md` with all organized keywords.
8.  **AEO Question Database (1-2 hours):** Create the `[Company]_AEO_Question_Database.md` with 50+ questions.
9.  **Executive Summary (30 mins):** Create the quick-reference `[Company]_SEO_Executive_Summary.md`.
10. **Actionable Content Plan Spreadsheet (2-3 hours):** Create the core deliverable `[Company]_Content_Plan.xlsx`.
11. **Content Plan User Guide (1 hour):** Create the `[Company]_Content_Plan_User_Guide.md`.
12. **Final Delivery (30 mins):** Package and deliver all initial assets.

### --- NEW ENHANCED PHASES --- 

### PHASE 13: Technical SEO Analysis (2-3 hours) - *For Existing Websites*

**Objective:** Analyze a Screaming Frog crawl export to identify and prioritize all technical SEO issues and create a detailed roadmap for fixes.

**Trigger:** This phase is activated if a Screaming Frog export file (`.csv` or `.xlsx`) is provided in the project files.

**Actions:**
1.  **Parse Screaming Frog Data:** Ingest and process the provided crawl data, focusing on key tabs like Internal, Response Codes, Page Titles, Meta Descriptions, H1, Canonicals, and Structured Data.
2.  **Categorize & Prioritize Issues:** Classify all identified issues into four priority levels:
    *   **P1 (Critical):** Site-breaking or major SEO-blocking issues (e.g., wrong noindex tags, critical 404s, redirect chains).
    *   **P2 (High):** Significant impact on performance or user experience (e.g., missing titles/H1s on key pages, slow pages).
    *   **P3 (Medium):** Important but less urgent issues (e.g., thin content, missing alt text).
    *   **P4 (Low):** Minor improvements and best practice recommendations.
3.  **Create Fix Roadmap:** Develop a week-by-week implementation timeline for all fixes, organized by priority.
4.  **Generate Detailed Instructions:** For each issue, write clear, step-by-step instructions that a developer or SEO technician can follow to resolve it.
5.  **Create Deliverable:** Build the `[Company]_Technical_SEO_Audit_and_Fixes.xlsx` spreadsheet.

### PHASE 14: Competitor Content Analysis (2-3 hours) - *For Existing Websites*

**Objective:** Analyze competitor content to determine their publishing frequency, identify content gaps, and establish a strategy for the client to outpace them.

**Actions:**
1.  **Analyze Competitor Blogs:** For the 4-6 competitors identified in Phase 4, navigate to their blog/articles section.
2.  **Extract Recent Content:** Systematically review the **last 10-15 posts** for each competitor.
3.  **Calculate Publishing Frequency:** Determine the average number of posts per week/month for each competitor.
4.  **Identify Content Gaps:** Analyze the topics, formats, and depth of competitor content to find what they are **not** covering.
5.  **Recommend Outpacing Strategy:** Based on the highest frequency found, recommend a publishing schedule for the client that is **1.5x to 2x greater**.
6.  **Create Deliverable:** Compile all findings into the `[Company]_Competitor_Content_Analysis.md` report.

### PHASE 15: Final Supercharged Delivery (30 mins)

**Objective:** Package and deliver the new technical and competitive analysis deliverables.

**Actions:**
1.  **Update Content Plan:** Integrate findings from the content gap analysis into the `[Company]_Content_Plan.xlsx` blog calendar.
2.  **Package All Files:** Organize all 15 phases of deliverables into a clean folder structure.
3.  **Deliver to User:** Send a final message to the user, attaching the new deliverables and explaining how they integrate with the foundational strategy.
