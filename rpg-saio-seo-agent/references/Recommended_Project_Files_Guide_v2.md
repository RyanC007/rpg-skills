# Recommended Project Files for Supercharged SEO/AEO Research Projects (v2)

**Purpose:** This guide outlines what files to upload to a Manus project to ensure consistent, high-quality SEO/AEO research output. It now includes specific requirements for running the **Supercharged Track** for clients with existing websites.

---

## 📁 REQUIRED FILES (Upload These for Every Client)

### 1. Company Knowledge Base
**Filename:** `[company_name]_knowledge_base.md` or `.json`

**What to Include:** This remains the most critical file. It should contain a comprehensive overview of the client's business, including their mission, services, target audience, unique value propositions, and geographic service areas.

**Why It's Critical:** This is the foundation for all strategic recommendations. The more detailed the knowledge base, the more tailored and effective the final strategy will be.

---

## 🚀 SUPERCHARGED TRACK REQUIREMENTS (For Existing Websites)

To activate the enhanced 15-phase methodology, which includes a full technical SEO audit and competitor content analysis, the following file **MUST** be provided in addition to the knowledge base.

### 2. Screaming Frog Crawl Export
**Filename:** `[company_name]_screaming_frog_export.csv` or `.xlsx`

**What to Include:** This should be a recent and complete crawl of the client's website from Screaming Frog SEO Spider. The "All Tabs" export is preferred, but individual exports of key tabs (Internal, Response Codes, Page Titles, etc.) are also acceptable.

**Why It's Critical:**
-   **This file is the trigger for the Supercharged track.** Without it, the AI will only perform the 12-phase foundational analysis.
-   It provides the raw data necessary for the AI to perform a deep technical SEO audit, identify issues, and create a prioritized fix roadmap.

---

## ✅ RECOMMENDED FILES (Upload for Better Results)

These files are not strictly required but will significantly improve the quality and specificity of the final deliverables for both tracks.

### 3. Brand Guidelines
**Filename:** `[company_name]_brand_guide.pdf` or `.md`
-   **Why It's Helpful:** Ensures the AI-generated content strategy aligns perfectly with the client's brand voice and messaging.

### 4. Existing Website Sitemap or Page List
**Filename:** `[company_name]_sitemap.xlsx` or `.pdf`
-   **Why It's Helpful:** Gives the AI a clear overview of the current site structure, preventing recommendations for duplicate content and helping to identify gaps more effectively.

### 5. Competitive Intelligence
**Filename:** `[company_name]_competitors.md`
-   **Why It's Helpful:** A simple list of known competitor URLs gives the AI a head start on the competitive analysis phase.

### 6. Market Research or Customer Insights
**Filename:** `[company_name]_market_research.pdf` or `.md`
-   **Why It's Helpful:** Helps the AI create content that resonates deeply with the target audience by addressing their specific pain points and motivations.

---

## 📊 FILE ORGANIZATION BEST PRACTICES

Use a clean and consistent folder structure within your Manus project to keep all assets organized.

```
/projects/[client-name]-[project-id]/
├── [client]_knowledge_base.md
├── [client]_screaming_frog_export.csv  (*** REQUIRED FOR SUPERCHARGED TRACK ***)
├── [client]_brand_guide.pdf
├── [client]_sitemap.xlsx
├── [client]_competitors.md
└── ... (other supporting files)
```

## 🎯 MINIMUM REQUIREMENTS FOR EACH TRACK

**Foundational Track (New Business):**
1.  ✅ Company Knowledge Base
2.  ✅ List of Services/Products
3.  ✅ Geographic Service Areas
4.  ✅ Target Customer Description

**Supercharged Track (Existing Business):**
1.  ✅ **All of the above, PLUS:**
2.  ✅ **Screaming Frog Crawl Export (`.csv` or `.xlsx`)**

**Bottom line:** The quality of the AI's output is directly proportional to the quality of the input files. For the Supercharged track, the Screaming Frog export is non-negotiable. A 1-hour investment in creating a thorough knowledge base and ensuring you have the correct technical files will result in a dramatically better strategy and higher client satisfaction.
