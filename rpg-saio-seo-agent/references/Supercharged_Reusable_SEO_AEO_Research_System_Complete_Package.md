## 🚀 Per-Client Workflow (Now with Two Tracks)

### Phase 1: Onboarding (30-60 minutes)
1.  Send the client intake questionnaire.
2.  **Crucially, ask: "Do you have an existing website?"**
3.  If yes, **request a Screaming Frog crawl export (`.csv` or `.xlsx`)**. Explain that this is necessary for the technical SEO audit.
4.  Create the comprehensive knowledge base as usual.

### Phase 2: Project Setup (10-15 minutes)
1.  Create a new Manus project.
2.  Upload all client files. **If the client has a website, ensure the Screaming Frog file is included.**
3.  Copy and customize the **Supercharged_SEO_AEO_Master_Prompt.md** into the project instructions.

### Phase 3: Research Execution (AI Time)
1.  Start the task in the Manus project.
2.  The AI will automatically detect the presence of the Screaming Frog file and execute the appropriate track:
    *   **No Screaming Frog file:** Executes the 12-phase Foundational Strategy.
    *   **Screaming Frog file present:** Executes the full 15-phase Supercharged Strategy.

### Phase 4: Review & Delivery (30-60 minutes)
1.  Review all deliverables for quality.
2.  If the Supercharged track was run, review the new **Technical SEO Audit** and **Competitor Content Analysis** deliverables carefully.
3.  Package all files for the client and deliver with a clear explanation of the findings and next steps.
