# LinkedIn Profile Analyzer & Growth Strategist

A comprehensive AI agent skill for analyzing LinkedIn profiles, evaluating content strategy, and generating strategic growth recommendations.

## Quick Start

To use this skill, simply provide a LinkedIn profile URL and specify what you need:

```
"Research this LinkedIn profile and analyze their content strategy: [URL]"
"Give me a strategic critique with recommendations for [LinkedIn URL]"
"Create a presentation summarizing the LinkedIn analysis for [URL]"
```

## What This Skill Does

This skill enables AI agents to:

1. **Research LinkedIn Profiles** - Automatically browse and collect 60+ posts from any LinkedIn profile
2. **Analyze Content Strategy** - Identify themes, engagement patterns, and what's resonating with audiences
3. **Generate Performance Reports** - Create comprehensive analysis documents with key insights
4. **Provide Strategic Recommendations** - Deliver actionable growth roadmaps using the Expert vs. Authority framework
5. **Build Executive Presentations** - Create professional 5-slide decks summarizing findings

## Core Framework: Expert vs. Authority

This skill uses a proven framework to assess LinkedIn presence:

**Expert** (Current State for most professionals):
- Shares knowledge through one-to-many broadcasting
- Content is informative but monologue-style
- Audience follows to learn

**Authority** (Target State for increased influence):
- Shapes industry conversations
- Acts as a hub that convenes and curates
- Builds community around ideas
- Elevates others alongside themselves

## Three Growth Levers

The skill recommends three high-leverage shifts to build authority:

1. **Broadcasting → Convening** - Turn posts into discussion hubs
2. **Solo Expert → Industry Hub** - Amplify other voices and curate insights
3. **Diversify Medium** - Add video and live formats for deeper connection

## Deliverables

### 1. Performance Analysis Report
A comprehensive Markdown document covering:
- Executive summary of LinkedIn presence
- Content strategy analysis (themes, formats, cadence)
- Audience engagement metrics and patterns
- Key strengths and differentiators

### 2. Strategic Growth Roadmap
A direct, actionable Markdown document with:
- Current state critique (Expert vs. Authority assessment)
- Three specific growth levers with 30/60/90-day action plans
- Tactical recommendations grounded in observed patterns

### 3. Executive Presentation (Optional)
A professional 5-slide deck featuring:
- Brand positioning overview
- Content strategy breakdown
- Engagement metrics visualization
- Key strengths summary
- Strategic recommendations
