# LinkedIn Profile Analyzer & Growth Strategist

## Overview

This skill enables AI agents to conduct comprehensive LinkedIn profile analysis, evaluate content strategy effectiveness, and generate strategic growth roadmaps. The agent researches LinkedIn profiles, analyzes posting patterns and audience engagement, and delivers both performance reports and actionable recommendations for increasing influence and authority.

## When to Use This Skill

Use this skill when:
- A user requests analysis of a LinkedIn profile's content strategy
- Someone wants to understand what's working (or not working) on a LinkedIn profile
- A user asks for recommendations to improve LinkedIn influence or authority
- You need to create a "wow report" analyzing someone's LinkedIn presence
- Strategic guidance is needed for LinkedIn thought leadership development

## Core Capabilities

This skill provides three primary outputs:

1. **LinkedIn Performance Analysis Report** - Comprehensive analysis of content themes, engagement patterns, and audience resonance
2. **Strategic Critique & Growth Roadmap** - Expert-level recommendations for increasing influence and authority
3. **Executive Presentation** - 5-slide deck summarizing key findings and strategic insights

## Workflow

### Phase 1: Profile Research & Data Collection

1. **Access the LinkedIn Profile**
   - Use browser tools to navigate to the provided LinkedIn profile URL
   - Focus on the `/recent-activity/all/` view to capture posts
   - Scroll through the feed to load 60+ posts (covering approximately 90 days)
   - Save findings incrementally to prevent data loss

2. **Capture Key Data Points**
   - Profile information (name, title, company, follower count)
   - Post content (titles, key themes, hashtags)
   - Engagement metrics (reactions, comments, reposts)
   - Visual content strategy (graphics, screenshots, media types)
   - Posting cadence and consistency

3. **Document Patterns**
   - Content themes and focus areas
   - Posting frequency and timing
   - Visual branding consistency
   - Engagement trends across different content types

### Phase 2: Strategic Analysis

1. **Evaluate Content Strategy**
   - Identify core content pillars and themes
   - Assess message consistency and positioning
   - Analyze content format effectiveness
   - Review visual branding quality

2. **Measure Audience Resonance**
   - Calculate average engagement rates
   - Identify top-performing content types
   - Assess comment quality and dialogue depth
   - Determine audience profile and composition

3. **Identify Strategic Gaps**
   - Compare current state to "authority" benchmarks
   - Spot opportunities for increased influence
   - Recognize format or medium limitations
   - Note community-building opportunities

### Phase 3: Deliverable Creation

1. **Performance Analysis Report** (Markdown document)
   - Executive Summary
   - Content Strategy Analysis
   - Audience Resonance & Engagement
   - Key Strengths & "Wow" Factor
   - Professional, paragraph-based format (not bullet points)

2. **Strategic Critique & Growth Roadmap** (Markdown document)
   - Current State Assessment (Expert vs. Authority framework)
   - Three High-Leverage Growth Levers
   - 90-Day Action Plan with specific tactics
   - Direct, actionable, no-BS recommendations

3. **Executive Presentation** (5-slide deck)
   - Slide 1: Brand Positioning Overview
   - Slide 2: Content Strategy Analysis
   - Slide 3: Audience Engagement Metrics
   - Slide 4: Key Strengths (Three Pillars)
   - Slide 5: The "Wow" Factor Summary
   - Use professional design (Swiss International Style recommended)

## Key Frameworks

### Expert vs. Authority Framework

**Expert:**
- Shares deep knowledge in their niche
- One-to-many broadcast model
- Content is a monologue
- Audience follows to learn what they know

**Authority:**
- Shapes industry conversations
- Hub model (convenes, curates, collaborates)
- Content is a dialogue
- Builds community around ideas
- Elevates others, not just themselves

### Three Levers for Authority Building

**Lever 1: Broadcasting → Convening**
- End posts with sharp, specific questions
- Engage actively in comments within first hour
- Run weekly polls for industry temperature checks

**Lever 2: Solo Expert → Industry Hub**
- Curate and comment on other experts' content
- Conduct text-based interviews
- Feature insightful follower comments

**Lever 3: Diversify Medium**
- Add short-form video (90-second insights)
- Host LinkedIn Live AMA sessions
- Show personal side to deepen trust

## Best Practices

### Research Phase
- Save findings to files after every 2-3 browser actions to prevent information loss
- Capture at least 60 posts to ensure comprehensive coverage
- Document both professional and personal content mix
- Note visual branding consistency across posts

### Analysis Phase
- Use the Expert vs. Authority framework as the core lens
- Focus on high-leverage opportunities, not minor tweaks
- Ground recommendations in observed data patterns
- Maintain Ryan's voice: direct, warm, grounded, no fluff

### Writing Phase
- Write in complete paragraphs, not bullet lists (except for action steps)
- Use professional, academic style for reports
- Use direct, conversational style for growth roadmap
- Avoid hyperbole and vague marketing jargon
- Make recommendations specific and actionable
