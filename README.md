# RPG Skills Library

**Ready, Plan, Grow! — AI Agent Skills**

This repository contains the official skill library for the RPG AI agent ecosystem, including Trinity (Ryan's Chief of Staff) and Scarlett (Marcela's Chief of Staff).

Skills are modular instruction sets that extend AI agent capabilities across content creation, research, SEO, business development, and more.

---

## Usage

This repository is designed for use with the [Manus AI](https://manus.im) platform. Skills are loaded automatically by Trinity and Scarlett at task startup.
